Assignment 2 — Embedded Control & Multiplexing

Focus: Interface MET1155 to an STM32 or FPGA. Use GPIO to output digits, toggle the demux select, and increment/decrement values.

  • Map GPIO pins to MET1155 headers and drive CD74HC4511.
  • Implement demux control with SN74LVC1G19 for multiplexing.
  • Write firmware to update digits and debounce inputs.
