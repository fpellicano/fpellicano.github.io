About MET2230

The MET2230 circuit board uses a Wheatstone bridge with an LDR feeding a 741 op‑amp configured as a differential amplifier; the gain is set by R1 above the op‑amp. The output drives an LED to indicate light intensity. Headers (P1 male, J1 female) and switch SW1 support onboard LDR mode or an external differential input, and easy connection to dev kits.
