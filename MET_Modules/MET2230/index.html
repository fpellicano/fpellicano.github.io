MET2230 – Op‑Amp Light Tracker Module

The MET2230 board is an educational tool for operational amplifiers, Wheatstone bridge circuits, and LDRs. A 741 op‑amp is wired as a differential amplifier across a bridge containing an LDR; the gain is set by resistor R1, and the output drives an LED whose brightness varies with light level. Applications include solar tracking, motion detection, and room‑light sensing.
