MET2230 – Op‑Amp Light Tracker Module

The MET2230 circuit board is an educational tool for operational amplifiers, Wheatstone bridge circuits, and light‑dependent resistors (LDRs). It uses a bridge containing an LDR feeding a 741 op‑amp configured as a differential amplifier; the gain is set by resistor R1 above the op‑amp.

The output drives an LED whose brightness varies with light level. The PCB provides three usage modes via labeled interfaces (SW1, P1, J1): use the onboard LDR bridge, bypass to an external differential source with SW1, or connect to microcontroller/FPGA headers compatible with common dev kits.
